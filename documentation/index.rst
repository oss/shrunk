Shrunk |release| Documentation
==============================

**Shrunk** is the official Rutgers University URL shortener. It allows users to
shorten long URLs into a more convenient Rutgers URL.

Shrunk is still in development. There are no guarantees of API stability and its
dependencies may change in the future.

Shrunk is written by Rutgers `Open System Solutions <http://oss.rutgers.edu>`_.

Issues
------
You can report a problem by opening up an issue on the shrunk `GitHub issues
page <https://github.com/oss/shrunk/issues>`_.

Contributing
------------
Shrunk is open source software. You can find the official project page `on
GitHub <http://github.com/oss/shrunk>`_. If you would like to contribute, fork
the project and send a pull request.
